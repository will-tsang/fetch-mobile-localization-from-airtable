# Helper script to convert csv to i18n locale file

## Setup

1. Prepare Airtable API Key

2. Install packages
    ```bash
    yarn
    ```

3. Prepare env variables
    ```bash
    cp .env.dist .env
    # fill in the env variables
    # Find Airtable Api Key in https://airtable.com/account
    ```

## Run script

```bash
update-locale.sh
```
